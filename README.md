# so-smart-stage — Dual-Layer Classroom Orchestrator

Raspberry Pi 5 orchestrator that combines a Google-Calendar–style schedule with V-JEPA video classification and drives three actions on a single shared event loop: ambient music that shifts with room phase, live lecture subtitles with on-shutdown summaries to Discord, and a wake-word voice assistant.
